Final Project A dan B

untuk tugas final project kelas A dan B membuat sistem perpustakaan dimana berisi 

User 
- melihat web informasi perpustakaan (homepage) 
- melakukan Pendaftaran akun dan login
- melihat katalog buku, melakukan pencarian buku serta filter bedasarkan kategori
- menampilkan popup informasi buku yang dicari mulai dari nama penerbit hingga nomor rak 
- melakukan sorting
- melakukan pengisian pengajuan peminjaman (dengan tanggal peminjaman dan pengembalian)
- jika peminjaman diterima maka status peminjaman menjadi disetujui dan stok buku menjadi berkurang, Jika ditolak maka akan memunculkan status pengajuan menjadi dibatalkan dan stok masih tetap
- buku tidak dapat dipinjam jika stok = 0 atau status stok = kosong
- bisa mengedit informasi diri

Admin
- CRUD Sistem, memasukkan stok buku yabg tersedia
- Melihat kategori buku 
- memverifikasi status buku menjadi dipinjam sesuai id peminjam serta stok buku akan berkurang jika buku tersebut dipinjam
- mengapprove dan menolak pengajuan peminjaman 
- melihat daftar siapa saja yang mengajukan permohonan 

Super Admin
- mendaftarkan admin
- melihat user yang terdaftar dan admin yang di daftarkan
- dapat mencabut (data admin tidak di hapus melainkan di hilangkan dari sistem) akses admin 
- melihat stok buku yang ada 

sistem dibangun menggunakan PHP,JS/JQuery,HTML menggunakan Single Page Application 

pengerjaan paling lambat tgl 22 Mei 2020

Pengumpulan tugas berupa link github dan presentasi ppt berupa informasi produk dan tata cara menggunakan produk yang dikirim ke masing masing koordinator tingkat

Final Project D dan C
untuk tugas final project kelas A dan B membuat sistem informasi mahasiswa dimana berisi website kampus
dan sistem berisi :
User 
- Mendaftar Sebagai Mahasiswa
- Melihat Informasi diri, daftar mahasiswa
- Melihat data Mahasiswa di salah satu kelas dan anggota pembimbing akademik (misal dosen b memiliki daftar mahasiswa yang di bimbing)
- Sort informasi (mis nim, nama, dsb)
- Melihat data diri dosen
- Bisa mengedit informasi diri
- mendaftarkan diri ke matakuliah (memiliki kuota minimal 5, jika melebihi ybs tidak bisa mendaftar)

Dosen
- Login Dosen
- Melihat Informasi diri, informasi daftar mahasiswa
- Sort informasi (mis nim, nama, dsb)
- membuat kelas dengan informasi kelas seperti daya tampung kelas, nama kelas, nomor id kelas, informasi jam, mengedit kelas dan menghilangkan kelas
- untuk kelas yang dihilangkan hanya memberikan status kelas tidak ada / tidak berlaku 
- melihat data mahasiswa yang mendaftar ke salah satu matakuliah

Staff / Admin
- mendaftarkan/registrasi dosen
- melihat informasi diri
- memverifikasi nama mahasiswa
- mencabut akses mahasiswa dengan label tertentu (mis meninggal dunia, belum membayar kuliah, dsb)
- mencabut akses dosen dengan label tertentu (mis mutasi, putus kontrak, dsb

sistem dibangun menggunakan PHP,JS/JQuery,HTML menggunakan Single Page Application (boleh menggunakan framework html atau js)
